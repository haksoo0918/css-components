# CSS Components

만든 이유는 페이지를 만들때 마다 만들었던 비슷한 CSS를 계속 반속해서 만드는것에,
또 했던 고민을 다시 반복하는 것 또한 효율이 너무 떨어진다 판단 했기 때문이다.

만들어 두고 가능한 복사해서 사용하기 위해, 변수는 중요하다 판단한 부분만 정의했다.

꼭 필요한 기능을 제하고선 JavaScript는 최대한 배제할 예정이다.
