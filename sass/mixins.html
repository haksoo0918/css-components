# clearfix

floating 해제

@mixin clearfix() {
  &::after {
    content: '';
    display: block;
    clear: both;
  }
}

.clearfix {
  @include clearfix;
}

# stretched link

확장 링크

@mixin stretched-link() {
  &::after {
    content: '';
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    z-index: 1;
    // box-sizing: border-box;
  }
}

.stretched-link {
  @include stretched-link;
}

# line-clamp

여러줄 문장 제한

  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: $num;
  overflow: hidden;
  // text-overflow: ellipsis;
